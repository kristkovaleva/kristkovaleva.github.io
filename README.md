# Kristina Kovaleva - Portfolio Website

A modern, responsive one-page portfolio website showcasing the professional experience and skills of Kristina Kovaleva, a 2D Animator.

## Features

- **Modern Design**: Clean, professional layout with gradient accents
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Hover effects and smooth transitions
- **Well-Organized Sections**: 
  - Professional Experience
  - Skills
  - Qualifications
  - Certificates
  - Languages

## Getting Started

Simply open `index.html` in your web browser to view the website. No build process or dependencies required.

## File Structure

```
kris-portfolio/
├── index.html      # Main HTML file
├── styles.css      # CSS styling
└── README.md       # This file
```

## Customization

You can easily customize the website by modifying:

- **Colors**: Edit the CSS variables in `styles.css` (lines starting with `--`)
- **Content**: Update the HTML in `index.html`
- **Fonts**: Change the Google Fonts import in the `<head>` section

## Browser Support

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
